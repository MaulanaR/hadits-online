Pertanyaan yang Sering Diajukan

Temukan jawaban untuk pertanyaan umum seputar aplikasi dan hadith.

Aplikasi & Penggunaan

Apakah aplikasi ini gratis?

Ya, aplikasi Sunnah.com sepenuhnya gratis untuk digunakan. Tidak ada biaya berlangganan atau fitur tersembunyi.

Bagaimana cara menyimpan hadith favorit?

Klik tombol hati (❤️) pada halaman hadith untuk menambahkan ke daftar favorit Anda. Favorit tersimpan di browser Anda.

Apakah data hadith akurat?

Hadith diambil dari sumber terpercaya seperti Kitab Suci (Shahih Bukhari, Muslim, Abu Dawud, dll). Setiap hadith mencantumkan nomor kitab dan referensi yang jelas.

Bagaimana cara mencari hadith?

Gunakan fitur pencarian dengan kata kunci, atau filter berdasarkan kitab, nomor hadith, atau bahasa (Arab/Indonesia).

Teknis & Bantuan

Browser apa yang didukung?

Aplikasi ini berfungsi optimal di browser modern: Chrome, Firefox, Safari, Edge, dan browser berbasis Chromium.

Apakah ada aplikasi mobile?

Saat ini aplikasi sudah responsif dan dapat digunakan dengan baik di perangkat mobile.

Bagaimana melaporkan bug?

Jika menemukan bug atau masalah, silakan laporkan melalui GitHub repository atau hubungi pengembang.

Apakah data saya aman?

Ya, semua data hadith disimpan lokal di browser Anda. Kami tidak mengumpulkan informasi pribadi.

Konten & Sumber

Dari mana sumber hadith?

Hadith diambil dari sumber terbuka yang dapat diakses umat, dengan tetap menghormati hak cipta dan referensi ke kitab asli.

Bagaimana cara kontribusi?

Anda dapat berkontribusi melalui:

  • Donasi: Untuk mendukung keberlangsungan aplikasi
  • Laporan Bug: Melalui GitHub repository
  • Saran: Untuk perbaikan dan pengembangan fitur
  • Sebar: Memberitahu orang lain tentang aplikasi ini

Apakah tersedia terjemahan bahasa lain?

Saat ini tersedia dalam bahasa Indonesia dan Arab. Untuk masa depan, kami merencanakan untuk menambahkan terjemahan dalam bahasa lain.
